\Environet\Sys\Plugins\TransportsHttpTransportExtended

Class HttpTransportExtended

Transport layer for http connections. Generalizes URL to also contain variables in square brackets, and files in a zip file separated by a pipe symbol. It also supports wildcards * and ? for file names in zip files.
