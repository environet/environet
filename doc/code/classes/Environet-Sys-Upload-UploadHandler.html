handleRequest()

handleRequest() : \Environet\Sys\General\Response|mixed

Handle the upload request.

It checks the auth state (username is presented in header, and an existing user), verifies the signature, verifies the xml with XSD schema, and processes the input. If an error occurs, an ErrorResponse XML will be generated (\Environet\Sys\Xml\CreateErrorXml).
