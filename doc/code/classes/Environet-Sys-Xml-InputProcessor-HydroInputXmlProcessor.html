process()

process() : mixed

Process the validated xml, and save the time series data in database Does the following steps: 1. Creates an UTC-time from the current timestamp.

  1. Finds the monitoring point id in xml. \Environet\Sys\Xml\InputProcessor\AbstractInputXmlProcessor::findMonitoringPoint()
  2. Iterates the observed properties in the xml, and updates time series for them.
  3. Inserts the results in the database. \Environet\Sys\Xml\InputProcessor\AbstractInputXmlProcessor::insertResults()
