run()

run(null  $flags = null) : int|null

For insert run must return the last inserted id, or null in case of error.

First it runs some easy check on the properties, and if is there any error, it won't run the query, but throws and exception After it it will build the query string based on the properties, and set the parameters bindings. It returns the fetchAll result of the statement.

Original flags
