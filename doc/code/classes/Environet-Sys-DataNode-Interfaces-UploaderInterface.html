doUpload()

doUpload() : bool

Do upload action.

DoUpload is a mandatory custom function that's responsible for triggering actual upload procedures. This has to be written by the user. This is called automatically every time an upload task is scheduled.

Returns

bool —

true on success, false otherwise.
